# F.A TEAM ELITE – Agenda de Missões

## 1. Configurar Firebase

1. Crie um projeto em https://console.firebase.google.com
2. Em **Configurações do Projeto > SDK Firebase para Web**, copie a configuração.
3. Substitua o conteúdo de `firebaseConfig.js` pelos seus dados.

## 2. Testar localmente

Abra `index.html` no navegador (precisa de conexão à internet para importar os módulos Firebase).

## 3. Cadastrar missões

Abra `admin.html`, preencha o formulário e salve.

## 4. Publicar no GitHub Pages

1. Crie um repositório (ex.: `fa-team-agenda`).
2. Envie todos os arquivos.
3. Em **Settings > Pages**, escolha Branch `main` e pasta `/root`.
4. Acesse: `https://seunome.github.io/fa-team-agenda/`

Pronto! Suas missões ficarão disponíveis e os operadores poderão confirmar presença.
